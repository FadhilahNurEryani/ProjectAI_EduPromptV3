1. DESKRIPSI PROJECT
================================================================================

EduPrompt AI adalah sebuah platform inovatif yang dirancang khusus untuk 
membantu guru dan pendidik Indonesia dalam membuat prompt AI yang berkualitas 
tinggi untuk keperluan pendidikan. Platform ini mengintegrasikan teknologi 
kecerdasan buatan terbaru untuk menghasilkan prompt yang efektif, relevan, dan 
disesuaikan dengan kurikulum Indonesia.

Dengan EduPrompt AI, pengguna dapat:
- Membuat template prompt custom yang profesional
- Menggenerate prompt berkualitas dengan bantuan AI (Groq LLM)
- Meningkatkan prompt sederhana menjadi prompt detail dan profesional
- Menyimpan dan mengelola prompt favorit mereka
- Menemukan ribuan template prompt siap pakai dalam berbagai kategori

Platform ini menggunakan model bahasa terkini Groq (llama-3.3-70b-versatile) 
untuk memastikan kualitas output yang tinggi dan response time yang cepat.

5. API ENDPOINTS LENGKAP
================================================================================

A. AUTHENTICATION ENDPOINTS

  POST /api/auth/register
  Purpose: Register user baru
  Request Body:
    {
      "email": "user@example.com",
      "password": "securePassword123",
      "name": "User Name"
    }
  Response:
    {
      "success": true,
      "message": "Registration successful",
      "user": { id, email, name }
    }
  Status: 201 Created / 400 Bad Request

  POST /api/auth/signin (NextAuth)
  Purpose: Login dengan credentials
  Request: Form-encoded atau JSON
    email: "user@example.com"
    password: "securePassword123"
  Response: JWT token di session

  GET /api/auth/session
  Purpose: Get current session info
  Response:
    {
      "user": { email, name },
      "expires": "ISO timestamp"
    }
  Status: 200 OK / 401 Unauthorized

B. TEMPLATE ENDPOINTS

  GET /api/templates
  Purpose: Get semua template atau filter
  Query Params:
    - category: string (optional)
    - difficulty: easy|intermediate|hard (optional)
    - search: string (optional)
    - limit: number (optional, default: 50)
    - offset: number (optional, default: 0)
  Response:
    {
      "success": true,
      "templates": [ { id, title, category, difficulty, ... } ],
      "total": number,
      "count": number
    }
  Status: 200 OK

  GET /api/templates/[id]
  Purpose: Get single template detail
  Response:
    {
      "success": true,
      "template": {
        "id": "uuid",
        "title": "Template Title",
        "description": "Detailed description",
        "category": "RPP",
        "difficulty": "intermediate",
        "variables": {
          "topic": "Topic description",
          "gradeLevel": "Grade level"
        },
        "promptTemplate": "Template string with {variables}",
        "exampleOutput": "Sample output",
        "instructions": "How to use",
        "tags": ["tag1", "tag2"],
        "estimatedTime": 30
      }
    }
  Status: 200 OK / 404 Not Found

  POST /api/templates/generate
  Purpose: Generate custom template dari deskripsi
  Auth: Required (JWT)
  Request Body:
    {
      "description": "Deskripsi template yang ingin dibuat",
      "subjectArea": "Subject (optional)",
      "gradeLevel": "Grade level (optional)"
    }
  Response:
    {
      "success": true,
      "template": {
        "title": "Generated Title",
        "description": "Generated description",
        "variables": { "var1": "description", ... },
        "promptTemplate": "Template string",
        "exampleOutput": "Example output",
        "instructions": "Usage instructions",
        "tags": ["tag1", "tag2"],
        "difficulty": "intermediate",
        "estimatedTime": 30
      }
    }
  Response Time: ~4-5 seconds (Groq API processing)
  Status: 200 OK / 401 Unauthorized / 500 Internal Server Error

C. PROMPT ENDPOINTS

  GET /api/prompts
  Purpose: Get user's saved prompts
  Auth: Required
  Query Params:
    - limit: number
    - offset: number
  Response:
    {
      "success": true,
      "prompts": [ { id, title, content, createdAt, ... } ],
      "total": number
    }
  Status: 200 OK / 401 Unauthorized

  POST /api/prompts
  Purpose: Save new prompt
  Auth: Required
  Request Body:
    {
      "title": "Prompt Title",
      "content": "Generated prompt content",
      "templateId": "uuid (optional)"
    }
  Response:
    {
      "success": true,
      "prompt": { id, title, content, createdAt, userId }
    }
  Status: 201 Created / 401 Unauthorized

  POST /api/prompts/enhance
  Purpose: Enhance simple prompt menjadi detail profesional (NEW FEATURE)
  Auth: Required
  Request Body:
    {
      "userInput": "Pembusukan tumbuhan",
      "context": "Untuk kelas 6 SD (optional)"
    }
  Response:
    {
      "success": true,
      "enhancedPrompt": "Detailed professional prompt here...",
      "originalInput": "Pembusukan tumbuhan",
      "context": "Untuk kelas 6 SD"
    }
  Response Time: ~1-2 seconds
  Status: 200 OK / 401 Unauthorized / 400 Bad Request
  
  Example Transformation:
    Input: "saya ingin belajar pembusukan tumbuhan"
    Output: "Jelaskan konsep pembusukan tumbuhan dengan fokus pada 
             proses penguraian bahan organik oleh mikroorganisme. 
             Sertakan tahapan-tahapan pembusukan, jenis-jenis 
             mikroorganisme yang terlibat, dan faktor-faktor yang 
             mempengaruhi kecepatan pembusukan. Gunakan contoh 
             konkret yang relevan dengan kehidupan sehari-hari..."

D. CATEGORIES ENDPOINT

  GET /api/categories
  Purpose: Get semua kategori template yang tersedia
  Response:
    {
      "success": true,
      "categories": [
        {
          "id": "uuid",
          "name": "RPP",

E. FAVORITES ENDPOINT

  GET /api/favorites

      "success": true,

  POST /api/favorites
  Purpose: Add template ke favorites
  Auth: Required
  Request Body:
    { "templateId": "uuid" }
  Response:
    { "success": true, "message": "Added to favorites" }
  Status: 201 Created / 401 Unauthorized

  DELETE /api/favorites/[templateId]
  Purpose: Remove dari favorites
  Auth: Required
  Status: 200 OK / 401 Unauthorized

6. DATABASE SCHEMA
================================================================================

MODEL User {
  id            String    @id @default(cuid())
  email         String    @unique
  password      String    (hashed dengan bcrypt)
  name          String?
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt

  relations:
    - savedPrompts (SavedPrompt[])
    - generatedPrompts (GeneratedPrompt[])
    - favorites (UserFavorite[])
}

MODEL Template {
  id                String    @id @default(cuid())
  title             String
  description       String
  category          String
  difficulty        String    (easy, intermediate, hard)
  variables         Json      (JSON object dengan variable descriptions)
  promptTemplate    String    (template dengan {variable} placeholders)
  exampleOutput     String
  instructions      String
  tags              String[]
  estimatedTime     Int       (minutes)
  createdAt         DateTime  @default(now())
  updatedAt         DateTime  @updatedAt

  relations:
    - userFavorites (UserFavorite[])
    - generatedPrompts (GeneratedPrompt[])
}

MODEL CustomPrompt {
  id                String    @id @default(cuid())
  title             String
  description       String
  variables         Json
  promptTemplate    String
  exampleOutput     String    (optional)
  instructions      String
  tags              String[]
  difficulty        String
  estimatedTime     Int
  createdAt         DateTime  @default(now())
  updatedAt         DateTime  @updatedAt
}

MODEL SavedPrompt {
  id                String    @id @default(cuid())
  title             String
  content           String    (full generated prompt)
  userId            String    @db.VarChar(255)
  user              User      @relation(fields: [userId], references: [id], onDelete: Cascade)
  templateId        String?   @db.VarChar(255)
  createdAt         DateTime  @default(now())
  updatedAt         DateTime  @updatedAt

  @@index([userId])
  @@index([templateId])
}

MODEL GeneratedPrompt {
  id                String    @id @default(cuid())
  templateId        String    @db.VarChar(255)
  template          Template  @relation(fields: [templateId], references: [id], onDelete: Cascade)
  userId            String    @db.VarChar(255)
  user              User      @relation(fields: [userId], references: [id], onDelete: Cascade)
  content           String    (full generated content)
  variables         Json      (used variable values)
  createdAt         DateTime  @default(now())

  @@index([userId])
  @@index([templateId])
}

MODEL UserFavorite {
  userId            String    @db.VarChar(255)
  user              User      @relation(fields: [userId], references: [id], onDelete: Cascade)
  templateId        String    @db.VarChar(255)
  template          Template  @relation(fields: [templateId], references: [id], onDelete: Cascade)
  createdAt         DateTime  @default(now())

  @@id([userId, templateId])
  @@index([templateId])
}

14. TROUBLESHOOTING
================================================================================

MASALAH: npm run dev error
SOLUSI:
  1. Clear node_modules: rm -rf node_modules
  2. Reinstall: npm install
  3. Clear cache: npm cache clean --force
  4. Check Node version: node --version (harus v20+)

MASALAH: Database connection error
SOLUSI:
  1. Verify PostgreSQL running: psql --version
  2. Check .env DATABASE_URL
  3. Test connection: psql <connection_string>
  4. Run migration: npx prisma migrate dev

MASALAH: Groq API errors
SOLUSI:
  1. Verify API key di .env
  2. Check Groq console (console.groq.com) untuk quota
  3. Check model availability: llama-3.3-70b-versatile
  4. Verify LLM_API_URL: https://api.groq.com/openai/v1/chat/completions

MASALAH: NextAuth errors
SOLUSI:
  1. Verify NEXTAUTH_SECRET is set
  2. Check NEXTAUTH_URL matches current domain
  3. Clear browser cookies
  4. Check database for Users table

MASALAH: TypeScript errors
SOLUSI:
  1. Run: npx tsc --noEmit
  2. Check tsconfig.json settings
  3. Rebuild: npm run build
  4. Clear .next folder: rm -rf .next
